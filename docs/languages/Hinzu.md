# Language _Hinzu_
**[GitHub](https://github.com/softlang/yas/blob/master/Hinzu)**
A trivial metadata/annotation language designed to facilitate basic documentation and semantic annotation of languages in the YAS repository.

## Language purposes
* Metadata definition
* Megamodeling
